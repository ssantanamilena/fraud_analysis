## fraud_analysis
# Projeto de Resolução Credit Card Fraud Detection -  Kaggle. 

Projeto de detecção de fraude em transações financeiras utilizando Random Forest.  
Foi aplicada amostragem devido ao grande volume de dados para otimização computacional.  
O modelo identifica padrões de fraude e analisa a importância das variáveis para suporte a decisões de negócio.
Link do download do DataSet https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud?resource=download
